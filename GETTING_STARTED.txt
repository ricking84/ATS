================================================================================
🎯 WHAT EACH PART DOES
================================================================================

FLOAT SENSOR (Status: Simulated)
├─ Shows when boat is in water
├─ Toggles water level display
└─ Updates every 100ms

GYRO SENSOR (Status: Simulated)
├─ Measures trailer angle/tilt
├─ Rotates 3D trailer model
├─ Shows angle in degrees
└─ Calculates "level" status

DISTANCE SENSORS (Status: Simulated)
├─ Left & Right sensors on boat
├─ Shows boat positioning
├─ Calculates alignment status
├─ Determines if boat is centered
└─ Moves boat in 3D view

📱 ACCESS FROM OTHER DEVICES
================================================================================

SAME COMPUTER: 
✓ http://localhost:5000

PHONE/TABLET ON SAME WiFi:
1. Find your computer's IP (Windows: ipconfig, Mac: ifconfig)
2. Open: http://<YOUR-IP>:5000
   Example: http://192.168.1.100:5000

INTERNET (Cloud Access):
1. Use ngrok: ngrok http 5000
2. Open the URL ngrok provides
3. Works on any device anywhere

================================================================================
🐛 COMMON ISSUES
================================================================================

"Dashboard won't load"
→ Make sure run.bat/run.sh is still running
→ Check if terminal shows errors
→ Try different browser (Chrome recommended)

"3D model not moving"
→ Open DevTools (F12) → Console tab
→ Look for JavaScript errors
→ Check if WebSocket connected

"Says 'Disconnected'"
→ Make sure backend is running
→ Check port 5000 is not blocked
→ Try stopping & restarting

"No data appearing"
→ If demo mode: DEMO_MODE=True should be on
→ Check browser console for errors
→ Reload the page

See README_SETUP.md for more troubleshooting
